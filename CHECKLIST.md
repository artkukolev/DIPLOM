## 🔧 Установка и локальное тестирование

### Шаг 1: Установите зависимости
```bash
npm install
```

**Проверки:**
- [ ] Команда завершилась без ошибок
- [ ] Создана папка `node_modules/`
- [ ] Создан файл `package-lock.json`

---

### Шаг 2: Запустите приложение локально
```bash
npm start
```

**Проверки:**
- [ ] Приложение открылось в браузере (http://localhost:3000)
- [ ] Нет красных ошибок в консоли браузера (F12)
- [ ] Страница в целом загрузилась за < 3 секунды

## 🌐 GitHub Pages / Vercel развёртывание

### GitHub Pages (вариант 1)

1. **Установите gh-pages:**
```bash
npm install --save-dev gh-pages
```
- [ ] gh-pages установлен

2. **Добавьте в package.json:**
- [ ] `"homepage": "https://USERNAME.github.io/ithub-journal"` (замените USERNAME)
- [ ] Scripts содержат `"predeploy"` и `"deploy"`

3. **Развёртывание:**
```bash
npm run deploy
```
- [ ] Команда завершилась успешно
- [ ] GitHub создал ветку `gh-pages`
- [ ] Settings → Pages указывает на `gh-pages` ветку

4. **Проверка:**
- [ ] Приложение доступно по URL
