## 🚀 Команды запуска

```bash
# Установка
npm install && cd frontend && npm install && cd ..

# Быстрый запуск (оба сервера)
npm run dev

# Раздельный запуск
npm start          # Backend на :5000
npm run client     # Frontend на :3000

# Сборка для production
npm run build
```
