# Руководство пользователя

## Вход в систему

1. Откройте приложение в браузере (http://localhost:3000)
2. Введите учётные данные:
   - Пользователь: `admin`
   - Пароль: `admin123`
3. Нажмите кнопку **Войти**

## Основные функции

### 1. Просмотр и поиск учащихся

- На левой панели отображается список всех учащихся
- Используйте поле поиска для поиска по имени или фамилии
- Щёлкните на учащегося, чтобы просмотреть его профиль и личные дела

### 2. Добавление нового учащегося

1. Нажмите кнопку **+ Добавить учащегося** на левой панели
2. Заполните форму с информацией:
   - **Основная информация** (имя, фамилия, отчество)
   - **Дата рождения и поступления**
   - **Класс и уровень**
   - **Информация о родителе/опекуне** (ФИ, телефон, email)
   - **Адрес проживания**
3. Нажмите **Сохранить**

### 3. Редактирование информации учащегося

1. Выберите учащегося из списка
2. В профиле учащегося нажмите на кнопку редактирования
3. Измените необходимые данные
4. Нажмите **Сохранить**

### 4. Удаление учащегося

1. На панели со списком учащихся найдите нужного учащегося
2. Нажмите кнопку **✕** справа от его имени
3. Подтвердите удаление в диалоговом окне

**⚠️ Внимание:** При удалении учащегося удаляются все его личные дела!

### 5. Управление личными делами

#### Просмотр личных дел

1. Выберите учащегося из списка
2. В правой части экрана откроется вкладка с личными делами этого учащегося
3. Все личные дела отображены в виде карточек с информацией:
   - Название и описание
   - Категория дела
   - Статус (Открыто, Закрыто, Архив)
   - Приоритет (🔴 Высокий, 🟡 Средний, 🟢 Низкий)
   - Срок выполнения

#### Создание нового личного дела

1. Выберите учащегося
2. В разделе личных дел нажмите **+ Добавить**
3. Заполните форму:
   - **Название** * (обязательно)
   - **Описание** (дополнительно)
   - **Категория** (например: "Поведение", "Учёба", "Здоровье")
   - **Приоритет** (Низкий, Средний, Высокий)
   - **Статус** (Открыто, Закрыто, Архив)
   - **Срок выполнения** (дата)
4. Нажмите **Сохранить**

#### Обновление личного дела

Щёлкните по личному делу, чтобы открыть его и отредактировать информацию.

#### Удаление личного дела

Нажмите кнопку **✕** на карточке личного дела и подтвердите удаление.

## Цветовая схема и интерфейс

Приложение использует гармоничную цветовую палитру:
- 🟣 **Фиолетовый** (#7C3AED) - основной цвет интерфейса
- 🩷 **Розовый** (#EC4899) - акценты и ссылки
- 🪻 **Лавандовый** (#C084FC) - дополнительные элементы

## Статусы личных дел

| Статус | Описание |
|--------|---------|
| **Открыто** 🔵 | Дело активно обрабатывается |
| **Закрыто** ✅ | Дело разрешено и завершено |
| **Архив** 📦 | Дело архивировано и больше не обрабатывается |

## Приоритеты

| Приоритет | Цвет | Использование |
|-----------|------|---------------|
| **🔴 Высокий** | Красный | Срочные, критичные вопросы |
| **🟡 Средний** | Жёлтый | Стандартные вопросы |
| **🟢 Низкий** | Зелёный | Некритичные, плановые вопросы |

## Выход из системы

Нажмите кнопку **🚪 Выход** в нижнем правом углу экрана.

## Советы и рекомендации

1. **Регулярно обновляйте информацию** - добавляйте новые личные дела и обновляйте статусы
2. **Используйте описание** - подробно описывайте проблему или вопрос в личном деле
3. **Устанавливайте сроки** - это поможет отслеживать сроки решения вопросов
4. **Категоризируйте дела** - используйте категории для удобства поиска и анализа
5. **Архивируйте завершённые дела** - это поможет неза загромождать текущий список

## Частые вопросы

**Q: Можно ли экспортировать данные?**
A: В текущей версии экспорт можно сделать через скопирование информации, экспорт в CSV планируется добавить.

**Q: Какие роли пользователей существуют?**
A: Два типа: **Администратор** (полный доступ) и **Учитель** (просмотр и редактирование личных дел).

**Q: Как восстановить удалённые данные?**
A: К сожалению, восстановление невозможно. Будьте осторожны при удалении!

**Q: Как защищены данные?**
A: Данные хранятся локально в защищённой базе данных SQLite с аутентификацией по токенам JWT.
