# 📚 IThub School - Электронный журнал

Полнофункциональное веб-приложение для управления электронным журналом школы. Разработано с использованием React с красивым дизайном в стиле glassmorphism и поддержкой темного режима.

## ✨ Особенности

- **🔐 Аутентификация**: Система входа с несколькими демо-аккаунтами (ученик, учитель, родитель, администратор)
- **📊 Личный кабинет**: Просмотр оценок, расписания и домашних заданий
- **📖 Управление оценками**: Полная таблица оценок с фильтрацией и сортировкой
- **📅 Расписание**: Интерактивное расписание занятий на неделю
- **📝 Домашние задания**: Управление и отслеживание выполнения заданий
- **👤 Профиль**: Личный кабинет с информацией об учащемся
- **⚙️ Панель администратора**: Управление пользователями, предметами и параметрами системы
- **🎨 Glassmorphism дизайн**: Современный интерфейс с размытыми стеклянными эффектами
- **🌓 Темный режим**: Удобный просмотр в любых условиях освещения
- **📱 Адаптивный дизайн**: Работает на десктопе, планшете и мобильном телефоне
- **🎯 Цветовая схема IThub**: Использованы фирменные цвета школы (лавандовый, фиолетовый, розовый)

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
npm install
```

### 2. Запуск в режиме разработки

```bash
npm start
```

Приложение откроется в браузере по адресу [http://localhost:3000](http://localhost:3000)

### 3. Сборка для продакшена

```bash
npm run build
```

Создаст оптимизированную сборку для развёртывания на GitHub Pages или другом хостинге.

## 🌐 Развёртывание

### На GitHub Pages

1. Обновите `package.json`:
```json
"homepage": "https://ваш-username.github.io/ithub-journal"
```

2. Установите `gh-pages`:
```bash
npm install --save-dev gh-pages
```

3. Добавьте скрипты в `package.json`:
```json
"deploy": "npm run build && gh-pages -d build",
"predeploy": "npm run build"
```

4. Развёртывайте:
```bash
npm run deploy
```

### На обычном хостинге

1. Соберите приложение:
```bash
npm run build
```

2. Загрузите содержимое папки `build/` на сервер

3. Настройте веб-сервер для маршрутизации всех запросов на `index.html`

## 🐛 Решение проблем

### Приложение не запускается
1. Убедитесь, что установлена корректная версия Node.js (14+)
2. Удалите папку `node_modules` и переустановите: `npm install`
3. Очистите кэш: `npm cache clean --force`

### Стили не загружаются
1. Убедитесь, что все CSS файлы импортированы
2. Проверьте пути к импортам
3. Откройте DevTools (F12) и проверьте консоль на ошибки

### Темный режим не работает
1. Убедитесь, что класс `dark-mode` применён к элементу `<div className="app">`
2. Проверьте, что CSS переменные для тёмного режима определены
