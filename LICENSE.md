## 🔍 Как правильно использовать

### Если вы копируете код

**ОБЯЗАТЕЛЬНО:**

1. **Включите файл LICENSE**
```bash
# Скопируйте файл в проект
cp LICENSE your-project/LICENSE
```

2. **Добавьте в README.md**
```markdown
## License

This project is based on IThub School by IThub Development Team.
Licensed under MIT License - see LICENSE file for details.
```

3. **Укажите в package.json**
```json
{
  "name": "my-app",
  "license": "MIT",
  "author": "Your Name",
  "description": "Based on IThub School React template"
}
```

### Если вы выкладываете на GitHub

1. **Скопируйте MIT LICENSE файл**
2. **Добавьте в Description:** "Based on IThub School Template"
3. **В README укажите:** Ссылку на исходный проект
4. **Лучше всего:** Сделайте Fork исходного репозитория
