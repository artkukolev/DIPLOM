### 1️⃣ Установите зависимости (1 команда)
```bash
npm install
```

## 💡 ПОЛЕЗНЫЕ СОВЕТЫ

### Фильтрация на странице Оценки
1. Выберите предмет из выпадающего списка
2. Выберите оценку (1-5)
3. Или введите текст в поле поиска
4. Таблица обновится автоматически

### Сортировка таблицы
Кликните на заголовок столбца (Ученик, Предмет, Оценка, Дата)

### Добавление оценки
1. Нажмите кнопку "➕ Выставить оценку"
2. Заполните все поля:
   - Ученик (выпадающий список)
   - Предмет (выпадающий список)
   - Оценка (1-5)
   - Примечание (опционально)
3. Нажмите "Добавить"

### Добавление задания
1. Нажмите кнопку "➕ Добавить задание"
2. Заполните все поля:
   - Предмет (выпадающий список)
   - Название (обязательно)
   - Описание (опционально)
   - Дедлайн (дата)
3. Нажмите "Добавить"

## 🐛 ЧТО ДЕЛАТЬ, ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

| Проблема | Решение |
|----------|---------|
| Файл не открывается | Убедитесь, что все 3 файла (app.html, style.css, script.js) в одной папке |
| Нет стилей | Перезагрузите браузер (Ctrl+Shift+R или Cmd+Shift+R) |
| Не работает логин | Используйте точный email из таблицы выше |
| Нет данных | Данные хранятся в браузере - проверьте, включен ли JavaScript |
| Потеряны данные | Очистите кеш браузера и откройте файл заново |
