# ⚡ Быстрый старт (5 минут)

## 1️⃣ Установка (2 минуты)

Откройте терминал в папке `DIPLOM` и выполните:

```bash
npm install && cd frontend && npm install && cd ..
```

## 2️⃣ Запуск (1 минута)

```bash
npm run dev
```

Приложение автоматически откроется на **http://localhost:3000**

## 3️⃣ Вход (30 секунд)

Используйте готовые учётные данные:

```
Пользователь: admin
Пароль: admin123
```

## 🆘 Проблемы?

1. Убедитесь, что Node.js установлен: `node --version`
2. Очистите npm кэш: `npm cache clean --force`
3. Удалите node_modules: `rm -rf node_modules frontend/node_modules`
4. Переустановите: `npm install && cd frontend && npm install && cd ..`
