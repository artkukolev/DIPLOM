## 📞 ПОДДЕРЖКА

Если что-то не работает:

1. **Проверить Node.js версию:**
   ```bash
   node --version  # должно быть 18+
   npm --version   # должно быть 9+
   ```

2. **Очистить и переустановить:**
   ```bash
   rm -rf node_modules package-lock.json
   npm install
   ```

3. **Перезапустить:**
   ```bash
   npm run dev
   ```

4. **Убить процессы на портах:**
   ```bash
   lsof -i :3000 -i :5001
   kill -9 <PID>
   ```
