## 💾 ГДЕ ХРАНЯТСЯ ДАННЫЕ?

Все данные сохраняются в **localStorage браузера**:
- localStorage закреплена за вкладкой в браузере
- Данные сохраняются между сеансами
- Максимум ~5-10 MB данных (достаточно для школы)

### Основные ключи:
- `app_users` - пользователи
- `app_students` - учащиеся
- `app_records` - личные дела
- `app_grades` - оценки
- `app_attendance` - посещаемость
- `app_timetable` - расписание
- `app_messages` - сообщения
- `app_events` - события
- `app_news` - новости
- `app_tickets` - тикеты поддержки
- `token` - текущий токен
- `user` - текущий пользователь

## ❓ FAQ

**Q: Потеряются ли мои данные?**
A: Нет! Они хранятся в localStorage и сохраняются между сеансами браузера.

**Q: Могу ли я использовать это на нескольких устройствах?**
A: Каждое устройство будет иметь свой набор данных (localStorage изолирован).

**Q: Что произойдет, если я очищу кэш браузера?**
A: Данные будут удалены, но вы всегда можете перезагрузить приложение для переинициализации.

**Q: Можно ли это развернуть на сервере?**
A: Да, просто скопируйте папку build на веб-сервер.

**Q: Почему нет бэкенда?**
A: На данный момент это был запрос - полностью независимое Frontend приложение без зависимостей от сервера.
