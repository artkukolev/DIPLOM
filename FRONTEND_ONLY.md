# 🎯 Frontend-Only школьное приложение (без бэкенда)

## ✅ Статус: ГОТОВО К ЗАПУСКУ

Это полностью независимое React приложение для управления личными делами учащихся. **Никакого бэкенда не требуется!**

---

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
cd frontend
npm install
```

### 2. Запуск приложения
```bash
npm start
```

Приложение откроется на `http://localhost:3000`

## 🔄 Как работает система без бэкенда

### Аутентификация
- Учетные данные проверяются против встроенного списка пользователей
- Токены генерируются на фронтенде
- Сессия хранится в localStorage

### Данные
- Все данные хранятся в localStorage браузера
- Каждый дата-объект имеет уникальный ID, сгенерированный на клиенте
- Операции CRUD выполняются через функции в `storage.js`
- Имитируется задержка сети для реалистичности (100ms)

### Поиск и фильтрация
- Все поиски выполняются на фронтенде в памяти
- Фильтрация работает по всем полям объектов
- Сортировка выполняется локально

## ⚙️ Функции API (все работают локально)

```javascript
// Аутентификация
authAPI.login(username, password)
authAPI.getCurrentUser()

// Учащиеся
studentsAPI.getAll(search, status, grade)
studentsAPI.getById(id)
studentsAPI.create(data)
studentsAPI.update(id, data)
studentsAPI.delete(id)

// Личные дела
recordsAPI.getByStudent(studentId)
recordsAPI.create(data)
recordsAPI.update(id, data)
recordsAPI.delete(id)

// Оценки
gradesAPI.getByStudent(studentId)
gradesAPI.getAverage(studentId)
gradesAPI.addGrade(data)
gradesAPI.deleteGrade(id)

// Посещаемость
attendanceAPI.getByStudent(studentId, date)
attendanceAPI.mark(data)

// Расписание
timetableAPI.getByClass(classId)
timetableAPI.getByDay(classId, dayOfWeek)

// События и новости
eventsAPI.getAll()
eventsAPI.create(data)
newsAPI.getAll()
newsAPI.create(data)

// Сообщения
messagesAPI.getAll()
messagesAPI.listConversations()
messagesAPI.sendMessage(conversationId, text)

// Поддержка
supportAPI.createTicket(data)
supportAPI.listTickets()

// Расписание задач
scheduleAPI.list()
scheduleAPI.create(data)
scheduleAPI.delete(id)
```

## 🐛 Устранение неполадок

### Приложение не загружает данные
- Проверьте, что localStorage включен в браузере
- Очистите кэш браузера (Ctrl+Shift+Delete)
- Перезагрузите страницу (Ctrl+R)

### Данные не сохраняются
- Проверьте, что браузер позволяет использовать localStorage
- Увеличьте лимит хранилища браузера если нужно
- Проверьте консоль браузера на ошибки (F12)

### Вход не работает
- Используйте правильные учетные данные (см. выше)
- Проверьте, что localStorage не переполнен
- Очистите localStorage: `localStorage.clear()` в консоли

## 📝 Заметки для разработчиков

### Добавление новой страницы
1. Создайте файл в `src/pages/NewPage.js`
2. Добавьте маршрут в `App.js`
3. Используйте API из `client.js` для работы с данными

### Добавление новой функции
1. Добавьте методы в подходящий объект в `src/utils/storage.js`
2. Экспортируйте функции качестве хранилища
3. Используйте в компонентах через API `client.js`

### Валидация данных
- Валидация выполняется на фронтенде перед сохранением
- Используйте встроенные HTML валидаторы
