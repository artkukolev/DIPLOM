# 📖 Быстрая справка - ВСЁ ЧТО НУЖНО ЗНАТЬ

## 🎯 Что это?

**Полностью рабочая веб-система** для управления личными делами учащихся в школе с красивым интерфейсом (фиолетовый, розовый, лавандовый).

## ⚡ Установка (30 секунд)

```bash
cd /Users/artemkukolev/Desktop/DIPLOM
npm install && cd frontend && npm install && cd ..
```

## 🚀 Запуск (10 секунд)

```bash
npm run dev
```

Откроется автоматически на **http://localhost:3000**

## 🔐 Вход (5 секунд)

```
Пользователь: admin
Пароль: admin123
```

## ✨ Что работает

- ✅ Полный список учащихся
- ✅ Добавление/редактирование/удаление учащихся
- ✅ Поиск по ученикам
- ✅ Создание личных дел
- ✅ Управление приоритетами (Высокий/Средний/Низкий)
- ✅ Статусы дел (Открыто/Закрыто/Архив)
- ✅ Красивый градиентный интерфейс

## 🆘 Если что-то не работает

```bash
# Очистить и переустановить
rm -rf node_modules frontend/node_modules
npm install && cd frontend && npm install && cd ..

# Проверить Node.js
node --version  # должно быть 14+

# Проверить портов (если заняты)
lsof -i :5000   # backend
lsof -i :3000   # frontend
```
