# ✨ Функции IThub School

Полное описание всех функций и возможностей приложения электронного журнала.

## 🎯 Общие функции

### 🔐 Аутентификация
- Экран входа с демо-аккаунтами
- Сохранение сессии в localStorage
- Выход из аккаунта
- Разные роли: ученик, учитель, родитель, администратор

### 🎨 Интерфейс
- **Glassmorphism дизайн** - современные стеклянные эффекты
- **Темный режим** - удобный просмотр в любое время суток
- **Адаптивный дизайн** - работает на всех устройствах
- **Плавные анимации** - красивые переходы между страницами
- **Цветовая схема IThub** - фирменные цвета (лавандовый, фиолетовый, розовый)

### 🧭 Навигация
- Липкая панель навигации сверху
- Быстрый доступ ко всем разделам
- Активная подсветка текущей страницы
- Отображение информации о пользователе
- Кнопка выхода и переключения темы

## 📚 Страницы и разделы

### 🏠 Главная панель (Dashboard)

**Описание:** Быстрый обзор вашей учебной деятельности

**Функции:**
- 📊 **Статистика**: средняя оценка, всего оценок, выполненные задания, посещаемость
- 👋 **Приветствие**: персонализированное сообщение при входе
- 📅 **Сегодняшние уроки**: расписание на день с кабинетами и учителями
- 📝 **Недавние события**: хронология выставления оценок и событий
- 💡 **Советы и рекомендации**: полезная информация для успешной учёбы

**Дополнительно:**
- Быстрые ссылки на расписание и оценки
- Информация о выполнении домашних заданий
- Система рекомендаций

---

### 📖 Оценки (Grades Page)

**Описание:** Полное управление и просмотр оценок

**Функции для учеников/родителей:**
- 📋 **Таблица оценок**: все оценки с датами, предметами и типами работ
- 🔍 **Фильтрация**: по предмету, значению оценки
- 🔎 **Поиск**: быстрый поиск по учителю или названию предмета
- 📊 **Сортировка**: по дате (новые), по оценке (выше), по предмету (А-Я)
- ⭐ **Статистика по предметам**: средняя оценка по каждому предмету

**Функции для учителей:**
- ✏️ **Добавление оценок**: форма для выставления новых оценок
- 🗑️ **Удаление оценок**: удаление ошибочно выставленных оценок
- ✐️ **Редактирование**: изменение существующих оценок

**Информация:**
- Средняя оценка студента
- Количество оценок
- Цветовые индикаторы (5 - зелёный, 4 - синий, 3 - жёлтый, 2 - красный)
- Личные данные (учитель, дата, тип работы)

---

### 📅 Расписание (Schedule Page)

**Описание:** Интерактивное расписание занятий на неделю

**Функции:**
- 📆 **Выбор дня**: быстрое переключение между днями недели (Пн-Пт)
- 🎓 **Полное расписание**: все уроки на день с временем, кабинетом, учителем
- ⏱️ **Временная шкала**: визуальная шкала времени с точками уроков
- 🎨 **Цветовые коды**: каждый предмет имеет свой цвет
- ☕ **Перемены**: отображение времени перемен между уроками

**Информация о каждом уроке:**
- Название предмета
- Время начала и длительность
- Номер кабинета
- ФИО учителя
- Продолжительность урока

**Дополнительно:**
- 📢 **Расписание звонков**: таблица с временем всех звонков
- 🎯 **Легенда цветов**: какой цвет какому предмету соответствует
- 📝 **Примечания**: информация об изменениях в расписании

---

### 📝 Домашние задания (Homework Page)

**Описание:** Управление домашними заданиями и отслеживание сроков

**Функции:**
- 📋 **Список заданий**: все домашние задания со статусом
- 🔍 **Фильтрация**: по статусу (в ожидании, выполнено, просрочено), по предмету
- 📊 **Статистика**: количество выполненных, в процессе, просрочено
- ✅ **Отметить выполненным**: изменение статуса задания
- 🗑️ **Удаление**: удаление выполненного или ненужного задания

**Информация о каждом задании:**
- Название задания
- Описание (полный текст задания)
- Предмет
- Учитель
- Срок выполнения
- Статус (с визуальными индикаторами)
- Количество оставшихся дней

**Статусы:**
- 🟡 **В ожидании** - задание нужно выполнить
- 🟢 **Выполнено** - задание выполнено в срок
- 🔴 **Просрочено** - срок истёк, задание не выполнено

**Дополнительно:**
- ⚠️ **Предупреждения**: выделение просроченных заданий
- 📌 **Предстоящие дедлайны**: сортировка по срокам
- 💡 **Советы**: рекомендации по управлению временем

---

### 👤 Профиль (Profile Page)

**Описание:** Личная информация и статистика пользователя

**Функции:**
- 👤 **Личные данные**: имя, email, телефон, адрес
- 🎓 **Образовательная информация**: класс, дата рождения, классный руководитель
- 📊 **Статистика**: оценки, задания, посещаемость, награды
- ✏️ **Редактирование**: возможность обновить контактную информацию
- 📝 **История событий**: недавние события (оценки, задания, посещение)

**Информация профиля:**
- Фото/аватар пользователя
- Полное имя
- Роль в системе
- Статус (активный пользователь)

**Статистика:**
- Всего оценок
- Средняя оценка
- Выполненные задания / Всего заданий
- Процент посещаемости
- Количество наград

**Для учеников:**
- Класс обучения
- Дата рождения
- Информация о классном руководителе
- Информация о школе

---

### ⚙️ Панель администратора (Admin Panel)

**Описание:** Управление пользователями, предметами и параметрами системы

**Функции:**

#### 👥 Управление пользователями
- Список всех пользователей с фильтрацией
- 🔍 **Поиск**: по email или имени
- 👤 **Добавление**: добавление новых пользователей
- ✏️ **Редактирование**: изменение информации пользователя
- 🔄 **Переключение статуса**: активирование/деактивирование пользователя
- 🗑️ **Удаление**: удаление пользователей из системы

**Информация о пользователе:**
- Email
- Имя
- Роль (ученик, учитель, родитель, администратор)
- Статус (активен/неактивен)
- Действия (редактирование, удаление)

#### 📚 Управление предметами
- Список всех предметов школы
- Карточки предметов с информацией об учителе
- ➕ **Добавление**: создание новых предметов
- ✏️ **Редактирование**: изменение информации о предмете
- 🗑️ **Удаление**: удаление предметов

#### ⚙️ Параметры системы
- 🔐 **Безопасность**: двухфакторная аутентификация, смена пароля
- 📧 **Уведомления**: настройки отправки уведомлений
- 📊 **Версия системы**: информация о версии
- 💾 **Резервная копия**: управление резервными копиями

#### 📊 Статистика
- Всего пользователей
- Количество учеников, учителей, родителей
- Здоровье системы
- Информация о базе данных

## 🎨 Дизайн и UX

### Glassmorphism эффекты
- Размытые фоны (blur filter)
- Полупрозрачные карточки
- Мягкие тени
- Плавные переходы

### Адаптивность
- **Desktop** (1200px+): полная версия всех элементов
- **Tablet** (768px-1199px): сжатая сетка, оптимизированная навигация
- **Mobile** (до 768px): мобильная версия с вертикальным скроллом

### Темный режим
- Все компоненты поддерживают тёмный режим
- Автоматическое переключение цветов
- Сохранение предпочтения пользователя
