## ❓ FAQ

### Как запустить?
```bash
npm install && npm run dev
```
Затем откройте http://localhost:3000

### Какой логин/пароль?
```
Логин: admin
Пароль: admin123
```

### Какой порт используется?
- Backend: 5001
- Frontend: 3000

### Где выполняется база данных?
```
backend/students.db (создаётся автоматически)
```

### Как изменить порт?
```bash
PORT=5002 npm start
```

### у меня ошибка при запуске
1. Удалить `node_modules` и `package-lock.json`
2. Запустить `npm install` заново
3. Проверить что Node.js и npm установлены
