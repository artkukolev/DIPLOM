# 🎓 Проект диплома: Система учёта личных дел учащихся

## 📋 Сводка

Создана **полнофункциональная веб-система** для управления личными делами учащихся в школьной образовательной организации с красивым интерфейсом в цветах **фиолетовый, розовый и лавандовый**.

## ✅ Что было создано

### 1️⃣ Backend (Node.js + Express)

```
backend/
├── server.js              - Главный файл сервера
├── database.js            - SQLite БД с 5 таблицами
├── routes/
│   ├── auth.js            - Вход, регистрация, проверка токена
│   ├── students.js        - CRUD для учащихся с поиском
│   ├── records.js         - CRUD для личных дел
│   └── documents.js       - Управление документами
├── middleware/
│   └── auth.js            - Проверка JWT токена
└── .gitignore
```

**Функции:**
- ✅ Аутентификация с JWT
- ✅ Полная система управления учащимися
- ✅ Создание, редактирование, удаление личных дел
- ✅ Управление документами
- ✅ История всех операций (аудит)
- ✅ Поиск и фильтрация

### 2️⃣ Frontend (React + Tailwind CSS)

```
frontend/
├── src/
│   ├── components/
│   │   ├── StudentForm.js         - Форма добавления/редактирования

│   │   ├── RecordsView.js         - Просмотр личных дел

│   ├── index.js                   - Точка входа
│   ├── index.css                  - Стили и анимации
│   └── tailwind.config.js         - Конфигурация Tailwind
└── public/
    └── index.html
```

**Особенности:**
- ✅ Красивый градиентный дизайн (фиолетовый → розовый → лавандовый)
- ✅ Адаптивный интерфейс для мобильных и ПК
- ✅ Интуитивная навигация
- ✅ Система приоритетов (Высокий, Средний, Низкий)
- ✅ Статусы личных дел (Открыто, Закрыто, Архив)
- ✅ Категоризация дел
- ✅ Поиск по ученикам

### 3️⃣ База данных (SQLite3)

Создана структурированная БД с 5 таблицами:

```sql
users               - Администраторы и учителя
students            - Информация об учащихся с контактами родителей
personal_records    - Личные дела с приоритетами и статусами
documents           - Прикрепленные документы
audit_log           - История всех операций
```

## 📈 API Endpoints (20+ маршрутов)

### Auth (3)
- POST /auth/login
- POST /auth/register
- GET /auth/me

### Students (5)
- GET /students
- POST /students
- GET /students/:id
- PUT /students/:id
- DELETE /students/:id

### Records (5)
- GET /records/student/:studentId
- POST /records
- GET /records/:id
- PUT /records/:id
- DELETE /records/:id

### Documents (3)
- GET /documents/record/:recordId
- POST /documents
- DELETE /documents/:id
