## 🚀 ДАЛЬНЕЙШИЕ ШАГИ

### Если хотите изменить данные:
1. Откройте `script.js`
2. Найдите `APP.data` (примерно на строке 50)
3. Отредактируйте демо-данные
4. Сохраните и обновите браузер (F5)

### Если хотите изменить дизайн:
1. Откройте `style.css`
2. Отредактируйте CSS классы или переменные в `:root`
3. Сохраните и обновите браузер (Ctrl+Shift+R)

### Если хотите добавить функции:
1. Откройте `script.js`
2. Добавьте новый метод в объект `APP`
3. Вызовите его из HTML элементов
